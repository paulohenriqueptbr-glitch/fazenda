Política de Privacidade

Atualizada em 07/06/2026

Resumo

O Controle Fazenda guarda os dados necessários para você registrar produção de leite, rebanho, lactações, gestações, medicações, relatórios e configurações da fazenda.

Dados coletados

  • Dados de conta, como nome, e-mail e senha gerenciada pelo Supabase Auth.
  • Dados da fazenda informados por você, como produção, animais, lactação, gestação, saúde e cotação do leite.
  • Dados salvos localmente no navegador para permitir uso offline e sincronização posterior.

Como usamos os dados

Usamos os dados para entregar o funcionamento do app, manter seu histórico, gerar relatórios, sincronizar informações e permitir exportação de backup em JSON.

Segurança

O app usa login por Supabase Auth. As tabelas do Supabase devem estar com RLS ativo para separar os dados por usuário usando a coluna user_id.

Compartilhamento

Os dados são processados por serviços usados para hospedar e operar o app, como Vercel e Supabase. Não vendemos dados dos usuários.

Backup e exclusão

Você pode exportar um backup manual em JSON pela aba Config. Para pedir correção ou exclusão de dados, entre em contato pelo suporte.
